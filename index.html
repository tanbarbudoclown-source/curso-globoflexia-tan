Después

Mayor seguridad, dominio de figuras básicas y avanzadas, ideas para cumpleaños, ventas y actividades creativas.

Contenido del curso

Estructura resumida para una landing clara y vendedora.

MÓDULO 1

Fundamentos

Qué es la globoflexia, materiales, tipos de globos, medidas, duración, helio, biodegradables y manipulación correcta.

MÓDULO 2

Técnicas

Inflado correcto, amarres, torciones, burbujas, herramientas y prácticas base para avanzar con seguridad.

Figuras

Figuras básicas y avanzadas con uno y dos globos, personajes y aplicaciones visuales para sorprender.

Uso en cumpleaños, cartillas tipo menú, estrategias iniciales para vender y bonus de evangelismo creativo.
